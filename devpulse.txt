## 5) Teach-Kel primer (the concepts YOU are learning)
Plain-English so you can teach it while you learn it:

- **API** = a way for your code to ask another company's server for data.
- **`HttpClient`** = the C# tool that sends that HTTP request.
- **JSON** = the format the answer comes back in. We map it into C# **models** (classes).
- **`async` / `await`** = "wait for the network without freezing the app."
- **Razor Pages** = HTML templates with C# that render on the server, then send a finished page to the browser.
- **Dependency injection** = "ask the app for a ready-made service" instead of `new`-ing everything by hand (`Program.cs` registers services).
- **`try` / `catch`** = if an API fails, show "couldn't load" instead of crashing.
- **Deploy** = publish the app to Azure so it has a real URL.

**Your one-liner for the room:** "An API is a waiter. `HttpClient` is you ordering. JSON is the plate. Razor puts what's on the plate onto the page."
